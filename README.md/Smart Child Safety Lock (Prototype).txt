Smart Child Safety Lock (Prototype)

Problem:
Young children can unintentionally unlock doors and leave the house unsupervised, creating serious safety risks.

Goal:
Design and build a microcontroller-based system that:
- Detects when a child is near a door
- Physically prevents the door from opening
- Allows manual adult override

Version 1 Scope:
- Ultrasonic sensor detects proximity at child height (~3–3.5 ft)
- ESP32 microcontroller processes detection logic
- Servo motor physically blocks door handle movement
- Manual push button for adult override
- LED indicator for testing and debugging

Hardware (Planned):
- ESP32 development board
- Ultrasonic distance sensor (HC-SR04)
- Servo motor
- Breadboard and jumper wires
- Push button
- LEDs
- Resistors
- USB cable / power supply

Software (Planned):
- Arduino IDE (ESP32 compatible)
- Serial Monitor for debugging
- Git/GitHub for version control

Status:
Project setup and documentation completed.
